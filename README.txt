This plugin provides the functionality of adding flights to any wordpress website, free and premium.

== Description ==

This is a plugin that allows wordpress user to have their flights website, simply make sure to have the prerequisite for optimal performance

A few notes about this plugin:

*   Get a MavisSMS Account
*   Get a TravelDen Api Key and Token
*   Get a Flutterwave API Public Key and Secret Key

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `gidi-flights.zip` to the `/wp-content/plugins/` directory
2. Extract `gidi-flights.zip` in the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
